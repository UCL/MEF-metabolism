# MEF-metabolism
These files relate to a study submitted to iScience exploring the impact of ras mutations in MEF cell lines. 
The code is functional but undergoing reworking to enhance readability and detailed documentation with tutorials is under development. Simulations should be run with dataRaw and dataKcat files in a folder that otherwise only has the simulation script.
